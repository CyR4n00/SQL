# 📋 Google スプレッドシート連携 セットアップガイド

Google スプレッドシートを DB マネージャーから読み込むには、
Google Cloud Console での設定が1回だけ必要です。

---

## どちらの認証方法を使うか？

| 方法 | 向いている人 | 手順の難しさ |
|------|------------|------------|
| **A. サービスアカウント** | 会社・チームで共有する | ★★☆ |
| **B. OAuth ログイン** | 個人の Google アカウントで使う | ★☆☆ |

---

## 共通手順：Google Cloud Console プロジェクト作成

### 1. Google Cloud Console を開く
👉 https://console.cloud.google.com/

### 2. プロジェクトを新規作成
- 画面上部の「プロジェクトを選択」→「新しいプロジェクト」
- プロジェクト名：`db-manager`（任意）→「作成」

### 3. API を有効化
左メニュー「APIとサービス」→「ライブラリ」から以下を検索して「有効化」：
- ✅ **Google Sheets API**
- ✅ **Google Drive API**

---

## A. サービスアカウント認証（会社・チーム向け）

### 4a. サービスアカウントを作成
1. 「APIとサービス」→「認証情報」→「認証情報を作成」→「サービスアカウント」
2. 名前を入力（例：`db-manager-sa`）→「作成して続行」→「完了」

### 5a. JSON キーをダウンロード
1. 作成したサービスアカウントをクリック
2. 「キー」タブ →「鍵を追加」→「新しい鍵を作成」→「JSON」→「作成」
3. ダウンロードされたファイルを `credentials/service_account.json` として保存

### 6a. スプレッドシートを共有
1. 対象のスプレッドシートを開く
2. 「共有」→ サービスアカウントのメールアドレス（`...@...iam.gserviceaccount.com`）を追加
3. 権限：「閲覧者」でOK

### 7a. アプリで認証
- 認証方法：「サービスアカウント」を選択
- 認証ファイル：`credentials/service_account.json`
- 「🔗 認証する」をクリック → ✓ 認証成功

---

## B. OAuth 認証（個人 Google アカウント向け）

### 4b. OAuth 同意画面を設定
1. 「APIとサービス」→「OAuth同意画面」
2. ユーザーの種類：「外部」→「作成」
3. アプリ名・メールを入力 → 「保存して次へ」×3 → 「ダッシュボードに戻る」

### 5b. OAuth クライアントIDを作成
1. 「認証情報」→「認証情報を作成」→「OAuth クライアント ID」
2. アプリの種類：「デスクトップアプリ」
3. 名前：`db-manager-desktop`→「作成」
4. 「JSONをダウンロード」→`credentials/oauth_client.json` として保存

### 6b. アプリで認証
- 認証方法：「OAuth（Googleアカウントでブラウザログイン）」を選択
- 認証ファイル：`credentials/oauth_client.json`
- 「🔗 認証する」をクリック
- ブラウザが開くので Google アカウントでログイン → 許可
- 一度ログインすれば次回はブラウザ不要

---

## 使い方

1. 認証後、スプレッドシートの URL を貼り付け
2. 「シート一覧取得」→ シートを選択
3. テーブル名を入力（省略可）→「📥 DB に取り込む」
4. あとは SQL タブで自由に操作できます！

---

## フォルダ構成

```
db_manager/
├── credentials/
│   ├── service_account.json  ← A の場合ここに置く
│   └── oauth_client.json     ← B の場合ここに置く
│   └── oauth_token.json      ← B の初回ログイン後に自動生成
```

## よくあるエラー

| エラー | 原因 | 対処 |
|--------|------|------|
| `File not found: service_account.json` | ファイルの場所が違う | `credentials/` フォルダに置く |
| `403 PERMISSION_DENIED` | スプレッドシートが共有されていない | 手順6aを確認 |
| `パッケージ不足` | ライブラリ未インストール | `pip install -r requirements.txt` |
| ブラウザが開かない | ポートがブロックされている | VPN・ファイアウォールを確認 |
